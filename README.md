leading-edge
========

前沿技术研究, 高性能Web构建研究

### 探究各种前沿技术，在企业级开发中的最佳实践。


## Get
```bash  
$ git clone https://github.com/starzou/leading-edge  
$ cd leading-edge  
```

## Install
```bash  
$ npm install && bower install
```

## Usage
```bash  
$ npm start
```
